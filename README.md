# study-python

这是一个用于学习与练习 Python 基础的仓库，包含选择题（含答案）以及面向 OJ 的练习题与参考答案。

## 仓库结构概览：

- `note/`
  - 存放~~课堂笔记与~~选择题（含答案）。
  - `note/old/` 目录为过时归档文件，仅作留存，不建议用于当前练习或参考。
- `practice/`
  - 面向在线评测（OJ）的练习题，按章节/知识点分目录（如 `01-输入输出`、`02-分支结构` 等）。
  - 每道题通常包含题目代码文件与参考答案，便于练习与对照。
- 项目根目录下的常用文件：
  - `helloworld.py`：示例文件
  - `LICENSE`：许可证
  - `pyproject.toml`：项目元信息（若需依赖管理请查看）

## 如何使用

- 运行某个练习题：在终端中使用 `python` 运行对应的 `.py` 文件，例如：

```bash
python practice/01-输入输出/1000.py
```

- 推荐 Python 版本：3.8 及以上。
